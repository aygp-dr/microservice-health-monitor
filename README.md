# microservice-health-monitor

Observability dashboard for microservices. Terminal UI built with [Bubble Tea](https://github.com/charmbracelet/bubbletea).

## Run

```bash
go run .
```

## Build

```bash
make build
./bin/microservice-health-monitor
```

## Controls

- `j/k`: navigate
- `Enter`: select/details
- `q`: quit
