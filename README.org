#+TITLE: microservice-health-monitor
#+AUTHOR: aygp-dr

Observability dashboard for microservices. Terminal UI built with [[https://github.com/charmbracelet/bubbletea][Bubble Tea]].

** Run

#+begin_src bash
go run .
#+end_src

** Build

#+begin_src bash
make build
./bin/microservice-health-monitor
#+end_src

** Controls

- =j/k=: navigate
- =Enter=: select/details
- =q=: quit
